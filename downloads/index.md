## 🛠️ 电子书制作

### 构建系统
本项目使用自动化构建系统生成电子书：
- **技术栈**: Pandoc + Node.js + Jekyll
- **样式优化**: 专为中文阅读优化
- **字体支持**: 完美支持中文字体显示
- **跨平台**: 支持 Windows、macOS、Linux

### 自定义构建
如果你想自己构建电子书：

```bash
# 克隆项目
git clone https://github.com/ShepherdWYQ/Beyond-the-Science-of-Reading.git

# 安装依赖
cd Beyond-the-Science-of-Reading
./scripts/setup-ebook-builder.sh

# 生成电子书
cd build && make all
```

详细说明请查看 [电子书构建指南]({{ '/EBOOK_BUILD.html' | relative_url }})
